# AI 메모 어플

## 앱 설명

기존의 앱
: 사용자는 메모를 적고 분류하고 일일이 찾아봐야 했다.

이 앱
: 사용자는 메모를 적기만 하면 이후의 과정은 모두 이 앱이 담당한다. 사용자가 메모를 작성할 때 앱에서는 관련된 태그를 1-3개를 뽑아내어 메모와 함께 저장한다. 메모의 내용, 태그, 사용자의 메모 패턴을 기반으로 이 앱은 사용자에게 다양한 편의를 제공한다. 이 앱은 3개의 섹션으로 구성되어 있다. 구글 로그인 기능을 지원한다.

## 섹션 설명

### 첫 번 째 섹션

첫 번 째 섹션은 사용자가 메모를 작성하는 부분이다. 사용자가 메모를 작성할 때 앱에서는 사용자가 작성하는 메모를 기반으로 1-3개의 태그를 제안한다. 사용자가 아무런 액션을 하지 않으면 메모는 제안된 태그들 중 첫 번 째 태그와 함께 저장되며, 사용자는 함께 저장하기를 원하는 태그를 1-3개까지 고르거나 추가로 1개까지 적을 수 있다. 첫 번 째 섹션의 중앙 부분에는 그동안 작성된 메모가 시간순으로 나와있다. (아래쪽으로 갈수록 최신 메모이다.) 첫 번 째 섹션의 우상단의 검색창에서는 메모를 검색할 수 있으며, 검색을 할 경우 해당 메모가 첫 번 째 섹션의 중앙에 오도록 찾아준다.

### 두 번 째 섹션

두 번 째 섹션은 사용자가 작성한 메모들을 시각적으로 확인할 수 있는 부분이다. 관계형을 클릭하면 옵시디언처럼 태그를 기반으로 관계형 트리 데이터가 그려지게 된다. 트리 데이터의 꼭짓점에 마우스를 가져다대면 해당 메모를 확인할 수 있게 된다. 그래프를 클릭하게 되면 사용자가 작성한 메모를 기반으로 다양한 통계 자료를 그려준다. (태그의 빈도, 메모의 작성 날짜 빈도 등)

### 세 번 째 섹션

세 번 째 섹션의 위쪽에는 사용자가 그동안 작성한 메모를 기반으로 다양한 AI 제안이 나온다. 세 번 째 섹션의 아래쪽에는 사용자가 그동안 작성한 메모를 기반으로 AI와 대화를 나를 수 있다.
